## 内容简介
本仓库主要是笔者根据中山大学物理与天文学院开设的《计算物理》课程整理的代码。并不完全按照老师的授课顺序与节奏。代码主要来源于三位老师的课程代码，尤其是来源于王龙老师的代码进行了额外的标注。

该课程主要涉及三个方向：


1. Monte Carlo方法、FFT及优化算法（张乐）
2. 机器学习及神经网络基本方法（李霄栋）
3. 常微分及偏微分方程求解（王龙）

除了笔者重新整理的代码之外，也会放上作业题，即`PartX_HW.ipynb`。在作业题中，会尤其针对一些`code`细节详细展开讲解，比如为什么在某些地方应该使用`lambda`函数等等.
